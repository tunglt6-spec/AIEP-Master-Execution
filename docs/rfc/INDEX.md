# AIEP — Requests for Comments (RFC)

Các RFC ghi lại những quyết định thiết kế quan trọng của AI Engineering Platform, kèm theo
động lực, thiết kế chi tiết, nhược điểm, các phương án và câu hỏi mở.

## Mục lục

| RFC | Tiêu đề | Trạng thái |
| --- | --- | --- |
| [0001](./RFC-0001-review-level-policy.md) | Review Level Policy (L1–L4) và Codex-L4-only Token-Preservation Guard | Accepted |
| [0002](./RFC-0002-ai-coding-agent.md) | AIEP × AI Coding Agent ("Idea → Code → Audit → Deploy") | Proposed |

## Quy ước

- Tên file: `RFC-NNNN-short-slug.md` (số đệm 0 ở đầu, tăng đơn điệu).
- Các giá trị trạng thái: `Draft` → `Accepted` / `Rejected` → `Superseded`.
- Mỗi RFC bao gồm: Tóm tắt, Động lực, Thiết kế chi tiết, Nhược điểm, Các phương án đã
  cân nhắc, Áp dụng / triển khai, và Câu hỏi mở.
